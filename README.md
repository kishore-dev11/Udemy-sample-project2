# Udemy Sample Projects

## HTML & CSS Flexbox Project

This project was created as part of my learning journey through a Udemy course. It focuses on building responsive layouts using **HTML**, **CSS**, and **Flexbox**.

## 🚀 Technologies Used

* HTML5
* CSS3
* Flexbox

## 🎯 What I Practiced

* Creating flexible layouts using Flexbox
* Using `justify-content` and `align-items`
* Building responsive designs
* Adding basic **animations** using `@keyframes`
* Applying **CSS transitions** for smooth UI effects

## ✨ Features

* Responsive layout
* Simple UI design inspired by Udemy
* Animated elements (zoom effect, hover effects)
* Clean and beginner-friendly structure

## 📌 Learning Outcome

This project helped me understand how to structure layouts efficiently and improve UI using animations and transitions.
